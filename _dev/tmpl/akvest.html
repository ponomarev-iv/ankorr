Аквест

Уважаемые господа!

Компания «Аквест» предлагает Вам универсальные огнезащитные  краски и составы для обработки металлических конструкций, воздуховодов, кабелей и  конструкций из дерева

1.Огнезащитная краска для стальных конструкций «Аквест-911». Огнезащитная краска «Аквест-911» предназначена для повышения предела огнестойкости несущих стальных конструкций до 90 минут, эксплуатируемых снаружи и внутри помещений с неагрессивной средой, не подвергающихся прямому воздействию воды и относительной влажности воздуха не более 80%. Соответствует требованиям пожарной безопасности, установленным в НПБ 236-97 «Огнезащитные составы для стальных. конструкций Общие требования. Метод определения огнезащитной эффективности».

2.Универсальная огнезащитная краска  «Аквест - 01». Огнезащитная краска «Аквест-01» предназначена для получения огнезащитного покрытия на  поверхностях  металлических  и  деревянных конструкций в  зданиях и  сооружениях различного назначения, в т.ч. на  объектах  гражданского  строительства.

3. Огнезащитная краска для воздуховодов «Аквест-01В». Огнезащитная краска «Аквест-01В» предназначена для повышения предела огнестойкости металлических воздуховодов, эксплуатируемых внутри сооружений промышленного и гражданского назначения.

4.Огнебиозащитный состав для дерева «Аквест-01Д». Огнебиозащитный состав предназначен для наружных и внутренних работ по обеспечению огнебиостойкости деревянных конструкций и сооружений. Расход для получения 1-ой группы огнезащитной эффективности 500 г/м2, для получения 2-ой группы - 300г/м2.

5.Краска акриловая воднодисперсионная «Аквест-01С»(предназначается для окрашивания путей эвакуации). Предназначена для наружных и внутренних работ, а также для окрашивания путей эвакуации в зданиях и сооружениях различного назначения. Пригодные для нанесения основания - бетон, кирпич, оштукатуренные или ранее окрашенные поверхности, минеральные плиты волокнистой структуры. Пригодна также для окрашивания  предварительно загрунтованного металла. Группы: горючести - Г1, воспламеняемости - В1, дымообразующая способность - Д1, токсичность - Т1.
